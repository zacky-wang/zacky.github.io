在做项目过程中，需要写一些数据库的文档说明，按照人工去完成总是会有错误出现，所以想写一个能够自动输出这个文档的小工具。

# 一、需求

这个工具主要有以下的功能：

- 支持大多数的关系型数据库
    - MySQL
    - PostgreSQL
- 支持多种启动形式
    - Web
    - Command
    - GUI(自带JRE)
- 支持多种输出格式
    - Word
    - Markdown
